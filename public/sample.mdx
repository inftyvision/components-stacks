## Code Examples

Here's a complex TypeScript example:

```typescript
interface Document {
  id: string;
  title: string;
  metadata: {
    author: string;
    created: Date;
    tags: string[];
  };
  content: {
    body: string;
    format: 'md' | 'html';
    excerpt?: string;
  };
}

class DocumentManager {
  private docs: Map<string, Document>;

  constructor() {
    this.docs = new Map();
  }

  async createDocument(doc: Omit<Document, 'id'>): Promise<string> {
    const id = crypto.randomUUID();
    this.docs.set(id, { ...doc, id });
    return id;
  }
}
```

# Best Practices

## Performance Tips

1. **Optimization**
   - Cache frequently used data
   - Use efficient algorithms
   - Implement lazy loading

2. **Code Quality**
   ```typescript
   // Good
   const getUserById = async (id: string): Promise<User> => {
     return await db.users.findUnique({ where: { id } });
   };

   // Bad
   const getUser = async (id) => {
     const users = await db.users.findMany();
     return users.find(u => u.id === id);
   };
   ```
