# Component Stack

A collection of React components for building modern documentation and content viewing experiences.

## Project Structure

```bash
components-stacks/
├── app/
│   ├── components/         # React components
│   │   └── AdvancedMdxViewer/  # MDX viewer component
│   └── styles/            # Global styles
└── src/
    └── docs/             # Documentation
```

## Components

### AdvancedMdxViewer

A documentation viewer that renders markdown content with pagination:

```typescript
interface AdvancedMdxViewerProps {
  content: string;        // Markdown content to display
  designSystem?: string;  // Optional design system
}
```

Features:
- Page-based navigation
- Responsive design
- Code block rendering
- Smooth transitions
- Automatic pagination
