# MDX Viewer

A simple markdown viewer component that renders MDX content with basic styling.

## Features

- Clean markdown rendering
- Code block support
- Basic styling
- Responsive design

## Usage

```typescript
import MdxViewer from '../components/MdxViewer';

function Documentation() {
  return (
    <MdxViewer content={`
      # Title
      
      Some content with \`inline code\` and:
      
      \`\`\`typescript
      function example() {
        return "Hello!";
      }
      \`\`\`
    `} />
  );
}
```

## Props

```typescript
interface MdxViewerProps {
  content: string;  // The markdown content to display
}
```

## Supported Markdown

1. **Headers**
   ```markdown
   # h1 Title
   ## h2 Title
   ### h3 Title
   ```

2. **Text**
   ```markdown
   Regular text
   *Italic*
   **Bold**
   ```

3. **Code**
   ```markdown
   `inline code`

   ```typescript
   // code block
   ```
   ```

4. **Lists**
   ```markdown
   - Unordered
   - List
     - Nested

   1. Ordered
   2. List
   ```

5. **Links**
   ```markdown
   [Link text](url)
   ```

6. **Blockquotes**
   ```markdown
   > Quote text
   ```
