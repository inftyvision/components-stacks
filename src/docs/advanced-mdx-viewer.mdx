# Advanced MDX Viewer

The Advanced MDX Viewer is a React component that renders markdown content with pagination and a clean interface.

## Features

- Page-based content navigation
- Responsive resizable container
- Clean code block rendering
- Smooth page transitions
- Automatic content pagination

## Usage

```typescript
import AdvancedMdxViewer from './components/AdvancedMdxViewer';

function Documentation() {
  return (
    <AdvancedMdxViewer
      content={`
# Page 1
Content for first page...

# Page 2
Content for second page...
      `}
    />
  );
}
```

## Props

```typescript
interface AdvancedMdxViewerProps {
  content: string;        // The markdown content to display
  designSystem?: string;  // Optional design system override
}
```

## Content Structure

The viewer automatically splits content into pages based on h1 headers:

```markdown
# First Page Title
This content will be on page 1...

## A Section
More content for page 1...

# Second Page Title
This will be on page 2...

## Another Section
More content for page 2...
```

## Supported Markdown

1. **Headers (h1-h3)**
   ```markdown
   # Main Title (h1)
   ## Section Title (h2)
   ### Subsection Title (h3)
   ```

2. **Text Formatting**
   ```markdown
   Regular text
   *Italic text*
   **Bold text**
   ```

3. **Lists**
   ```markdown
   - Unordered list
   - Another item
     - Nested item

   1. Ordered list
   2. Second item
      - Mixed nesting
   ```

4. **Code**
   ````markdown
   Inline `code` and:

   ```typescript
   function example() {
     return "Hello!";
   }
   ```
   ````

5. **Blockquotes**
   ```markdown
   > This is a quote
   > It can span multiple lines
   ```

6. **Links**
   ```markdown
   [Link text](https://example.com)
   ```

## Navigation

- Use the Previous/Next buttons to move between pages
- Current page and total pages are shown in the header
- Smooth transitions between pages
- Resizable container for comfortable reading
