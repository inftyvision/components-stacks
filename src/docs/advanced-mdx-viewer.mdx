# Advanced MDX Viewer

Welcome to the Advanced MDX Viewer documentation. This component extends our base MDX viewer with enhanced features and capabilities.

## Overview

The Advanced MDX Viewer builds upon our standard MDX viewer, adding:

- Table of Contents navigation
- Search functionality
- Dark/Light theme switching
- Zoom controls for code blocks
- Copy to clipboard functionality
- Keyboard shortcuts
- Collapsible sections
- Progress indicator
- Print mode
- Mobile-optimized view

## Installation

```bash
npm install @mdx-js/react react-markdown remark-gfm
```

## Component Structure

```typescript
interface AdvancedMdxViewerProps {
  content: string;
  theme?: 'light' | 'dark' | 'system';
  initialPage?: number;
  showToc?: boolean;
  showProgress?: boolean;
  enableSearch?: boolean;
  enableKeyboardNav?: boolean;
  onPageChange?: (page: number) => void;
  onThemeChange?: (theme: string) => void;
}
```

## Features

### Table of Contents

The ToC provides:
- Automatic generation from headers
- Collapsible sections
- Active section highlighting
- Smooth scrolling
- Sticky positioning

### Search Functionality

Full-text search with:
- Instant results
- Highlight matches
- Search history
- Keyboard navigation
- Advanced filters

### Theme Support

Comprehensive theming:
- Light/Dark modes
- System preference detection
- Custom color schemes
- Persistent preferences
- Smooth transitions

### Code Blocks

Enhanced code display:
- Syntax highlighting
- Line numbers
- Copy button
- Language detection
- Zoom controls
- Word wrap toggle

### Keyboard Navigation

Intuitive shortcuts:
- `←/→` - Previous/Next page
- `Ctrl+F` - Search
- `Esc` - Close overlays
- `+/-` - Zoom in/out
- `Ctrl+P` - Print mode

### Mobile Experience

Optimized for mobile:
- Responsive layout
- Touch gestures
- Collapsible navigation
- Readable typography
- Offline support

## Usage Examples

### Basic Setup

```typescript
import AdvancedMdxViewer from '../components/AdvancedMdxViewer';

function Documentation() {
  return (
    <AdvancedMdxViewer
      content={content}
      theme="system"
      showToc={true}
      showProgress={true}
      enableSearch={true}
      enableKeyboardNav={true}
    />
  );
}
```

### Custom Theme

```typescript
<AdvancedMdxViewer
  content={content}
  theme="dark"
  onThemeChange={(theme) => {
    console.log(`Theme changed to: ${theme}`);
    // Update system theme
  }}
/>
```

### With Event Handlers

```typescript
<AdvancedMdxViewer
  content={content}
  initialPage={2}
  onPageChange={(page) => {
    console.log(`Navigated to page: ${page}`);
    // Update URL or state
  }}
/>
```
